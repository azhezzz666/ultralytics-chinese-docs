# 为 Ultralytics 开源项目做贡献

欢迎！我们非常高兴您考虑为我们的 [Ultralytics](https://www.ultralytics.com/) [开源](https://github.com/ultralytics)项目做贡献。您的参与不仅有助于提升我们仓库的质量，还能惠及整个[计算机视觉](https://www.ultralytics.com/glossary/computer-vision-cv)社区。本指南提供了清晰的准则和最佳实践，帮助您快速入门。

## 🤝 行为准则

为确保每个人都能在一个友好和包容的环境中工作，所有贡献者必须遵守我们的[行为准则](https://docs.ultralytics.com/help/code-of-conduct/)。**尊重**、**友善**和**专业**是我们社区的核心价值观。

## 🚀 通过 Pull Request 贡献

我们非常感谢以 Pull Request (PR) 形式提交的贡献。为使审核过程尽可能顺利，请遵循以下步骤：

1. **Fork 仓库：** 首先将相关的 Ultralytics 仓库 Fork 到您的 GitHub 账户。
2. **创建分支：** 在您 Fork 的仓库中创建一个新分支，使用清晰、描述性的名称。
3. **进行更改：** 实现您的改进或修复。
4. **测试更改：** 在提交之前，在本地测试您的更改。
5. **提交更改：** 使用简洁且描述性的提交消息提交您的更改。
6. **创建 Pull Request：** 从您的分支向原始仓库的 main 分支提交 Pull Request。

## 📜 许可证

Ultralytics 的仓库使用 GNU Affero 通用公共许可证 v3.0 (AGPL-3.0)。此许可证促进软件开发中的开放性、透明性和协作改进。

## 🎉 结语

感谢您有兴趣为 Ultralytics 开源 YOLO 项目做贡献。您的参与对于塑造我们软件的未来和建立一个充满活力的创新与协作社区至关重要。
