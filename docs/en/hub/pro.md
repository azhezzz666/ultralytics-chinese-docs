# Ultralytics HUB Pro

[Ultralytics HUB](https://www.ultralytics.com/hub) 提供 Pro 计划作为月度或年度订阅，旨在通过高级功能和能力增强您的 AI 开发体验。

## Pro 计划优势

Pro 计划提供即将推出功能的早期访问权限，并包含多项增强优势：

- **扩展存储**：200GB 存储空间，相比免费计划的 20GB
- **云训练**：访问 [Ultralytics 云训练](./cloud-training.md)进行一键模型训练
- **高级 API 访问**：
    - 访问我们的[专用推理 API](./inference-api.md#专用推理-api)
    - 提高[共享推理 API](./inference-api.md#共享推理-api) 的速率限制
- **团队协作**：[团队协作](./teams.md)功能以简化工作流程
- **月度积分**：每月 20 美元积分（或每年 200 美元）用于支付训练费用

## 如何升级

您可以按照以下步骤从[设置](https://hub.ultralytics.com/settings)页面的[计费与许可](https://hub.ultralytics.com/settings?tab=billing)选项卡升级到 Pro 计划：

1.  点击**升级**按钮

    ![Ultralytics HUB 设置页面计费与许可选项卡截图，箭头指向升级按钮](https://github.com/ultralytics/docs/releases/download/0/ultralytics-hub-settings-upgrade-button.avif)

2.  选择 Pro 计划

    ![Ultralytics HUB 升级对话框截图，箭头指向选择计划按钮](https://github.com/ultralytics/docs/releases/download/0/hub-pro-upgrade-select-plan.avif)

    !!! tip "提示"

        如果选择年度 Pro 计划，您可以节省 20%。

        ![Ultralytics HUB 升级对话框截图，箭头指向节省 20% 开关和选择计划按钮](https://github.com/ultralytics/docs/releases/download/0/hub-pro-upgrade-save-20-toggle.avif)

3.  在结账时填写您的详细信息

    ![Ultralytics HUB 结账截图，箭头指向保存支付信息以供将来购买的复选框](https://github.com/ultralytics/docs/releases/download/0/hub-pro-upgrade-save-payment-info.avif)

    !!! tip "提示"

         我们建议勾选复选框以保存您的支付信息以供将来购买，便于更轻松地为账户余额充值。

4.  完成支付流程

    ![Ultralytics HUB 支付成功对话框截图](https://github.com/ultralytics/docs/releases/download/0/payment-successful-dialog.avif)

## 管理您的账户余额

账户余额用于支付 [Ultralytics 云训练](./cloud-training.md)资源。使用轮次训练方法时，您的账户余额在每个[轮次](https://www.ultralytics.com/glossary/epoch)后扣除，您需要至少 5.00 美元的余额才能开始训练。

### 如何充值余额

1. 在计费与许可选项卡中点击**充值**按钮

    ![Ultralytics HUB 设置页面计费与许可选项卡截图，箭头指向充值按钮](https://github.com/ultralytics/docs/releases/download/0/hub-pro-account-balance-top-up-button.avif)

2. 设置您要添加到余额的金额

    ![Ultralytics HUB 结账截图，箭头指向更改金额按钮](https://github.com/ultralytics/docs/releases/download/0/hub-pro-account-balance-change-amount.avif)

3. 完成支付流程

    ![Ultralytics HUB 支付成功对话框截图](https://github.com/ultralytics/docs/releases/download/0/payment-successful-dialog-1.avif)
