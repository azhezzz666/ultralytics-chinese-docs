# Ultralytics HUB 团队

我们很高兴向您介绍 [Ultralytics HUB](https://www.ultralytics.com/hub) 为 [Pro](./pro.md) 用户提供的全新团队功能！

在这里，您将学习如何管理团队成员、无缝共享资源以及在各种项目上高效协作。

!!! note "注意"

    由于这是一项新功能，我们仍在开发和完善中，以确保它满足您的需求。

## 创建团队

!!! note "注意"

    您需要[升级](./pro.md#如何升级)到 [Pro 计划](./pro.md)才能创建团队。

    ![Ultralytics HUB 设置页面团队选项卡截图，箭头指向升级按钮](https://github.com/ultralytics/docs/releases/download/0/hub-create-team-settings-page.avif)

点击[设置](https://hub.ultralytics.com/settings)页面中的**团队**选项卡导航到[团队](https://hub.ultralytics.com/settings?tab=teams)页面，然后点击**创建团队**按钮。

![Ultralytics HUB 团队页面截图，箭头指向创建团队按钮](https://github.com/ultralytics/docs/releases/download/0/ultralytics-hub-create-team-button.avif)

此操作将触发**创建团队**对话框。

在_团队名称_字段中输入团队名称或保留默认名称，然后一键完成团队创建。

您还可以选择为团队添加描述和独特图像，增强其在[团队](https://hub.ultralytics.com/settings?tab=teams)页面上的可识别性。

当您对团队配置满意时，点击**创建**。

![Ultralytics HUB 创建团队对话框截图，箭头指向创建按钮](https://github.com/ultralytics/docs/releases/download/0/hub-create-team-dialog.avif)

团队创建后，您可以从[团队](https://hub.ultralytics.com/settings?tab=teams)页面访问它。

![Ultralytics HUB 团队页面截图，箭头指向其中一个团队](https://github.com/ultralytics/docs/releases/download/0/hub-teams-page-arrow-pointing-to-team.avif)

## 编辑团队

导航到[团队](https://hub.ultralytics.com/settings?tab=teams)页面，打开要编辑的团队的团队操作下拉菜单，点击**编辑**选项。此操作将触发**更新团队**对话框。

![Ultralytics HUB 团队页面截图，箭头指向其中一个团队的编辑选项](https://github.com/ultralytics/docs/releases/download/0/hub-edit-team-1.avif)

对团队应用所需的修改，然后点击**保存**确认更改。

![Ultralytics HUB 更新团队对话框截图，箭头指向保存按钮](https://github.com/ultralytics/docs/releases/download/0/hub-update-team-save-button.avif)

## 删除团队

导航到[团队](https://hub.ultralytics.com/settings?tab=teams)页面，打开要删除的团队的团队操作下拉菜单，点击**删除**选项。

![Ultralytics HUB 团队页面截图，箭头指向其中一个团队的删除选项](https://github.com/ultralytics/docs/releases/download/0/hub-delete-team-option.avif)

!!! warning "警告"

    删除团队后，团队无法恢复。

## 邀请成员

导航到要添加新成员的团队的团队页面，点击**邀请成员**按钮。此操作将触发**邀请成员**对话框。

![Ultralytics HUB 团队页面截图，箭头指向邀请成员按钮](https://github.com/ultralytics/docs/releases/download/0/hub-invite-member-button.avif)

输入新成员的电子邮件并选择角色，然后点击**邀请**。

![Ultralytics HUB 邀请成员对话框截图，箭头指向邀请按钮](https://github.com/ultralytics/docs/releases/download/0/hub-invite-member-dialog.avif)

![Ultralytics HUB 团队页面截图，显示已邀请的新成员](https://github.com/ultralytics/docs/releases/download/0/hub-invite-member-3.avif)

??? tip "提示"

    您可以在新成员接受邀请之前取消邀请。

    ![Ultralytics HUB 团队页面截图，箭头指向其中一个成员的取消邀请选项](https://github.com/ultralytics/docs/releases/download/0/hub-invite-member-cancel-invite.avif)

新成员接受邀请后，**待处理**状态会消失。

![Ultralytics HUB 团队页面截图，显示两个成员](https://github.com/ultralytics/docs/releases/download/0/team-page-two-members.avif)

??? tip "提示"

    您可以随时更新成员的角色。

    **管理员**角色允许邀请和移除成员，以及移除共享的数据集或项目。

    ![Ultralytics HUB 团队页面截图，箭头指向其中一个成员的更改角色选项](https://github.com/ultralytics/docs/releases/download/0/hub-invite-member-change-role.avif)
