## 部署集成

- [Axelera](axelera.md)：探索 Metis 加速器和 Voyager SDK，在高效的边缘推理中运行 Ultralytics 模型。

- [CoreML](coreml.md)：CoreML 由 [Apple](https://www.apple.com/) 开发，是一个专为在 iOS、macOS、watchOS 和 tvOS 应用程序中高效集成机器学习模型而设计的框架，使用 Apple 硬件进行有效和安全的[模型部署](https://www.ultralytics.com/glossary/model-deployment)。

- [ExecuTorch](executorch.md)：由 [Meta](https://about.meta.com/) 开发，ExecuTorch 是 PyTorch 在边缘设备上部署 Ultralytics YOLO 模型的统一解决方案。

- [Gradio](gradio.md)：使用 Gradio 部署 Ultralytics 模型，实现实时、交互式的目标检测演示。

- [MNN](mnn.md)：由[阿里巴巴](https://www.alibabagroup.com/)开发，MNN 是一个高效轻量的深度学习框架。它支持深度学习模型的推理和训练，在设备端推理和训练方面具有行业领先的性能。

- [NCNN](ncnn.md)：由[腾讯](http://www.tencent.com/)开发，NCNN 是一个专为移动设备定制的高效[神经网络](https://www.ultralytics.com/glossary/neural-network-nn)推理框架。它能够将 AI 模型直接部署到应用程序中，优化各种移动平台的性能。

- [Neural Magic](neural-magic.md)：利用量化感知训练 (QAT) 和剪枝技术优化 Ultralytics 模型，获得卓越的性能和更小的模型尺寸。

- [ONNX](onnx.md)：由 [Microsoft](https://www.microsoft.com/) 创建的开源格式，用于促进 AI 模型在各种框架之间的传输，增强 Ultralytics 模型的通用性和部署灵活性。

- [OpenVINO](openvino.md)：Intel 的工具包，用于在各种 Intel CPU 和 GPU 平台上高效优化和部署[计算机视觉](https://www.ultralytics.com/glossary/computer-vision-cv)模型。

- [PaddlePaddle](paddlepaddle.md)：由[百度](https://www.baidu.com/)开发的开源深度学习平台，PaddlePaddle 能够高效部署 AI 模型，专注于工业应用的可扩展性。

- [Rockchip RKNN](rockchip-rknn.md)：由 [Rockchip](https://www.rock-chips.com/) 开发，RKNN 是一个专为 Rockchip 硬件平台（特别是其 NPU）优化的专用神经网络推理框架。它促进了 AI 模型在边缘设备上的高效部署，实现实时应用中的高性能推理。

- [Seeed Studio reCamera](seeedstudio-recamera.md)：由 [Seeed Studio](https://www.seeedstudio.com/) 开发，reCamera 是一款专为实时计算机视觉应用设计的先进边缘 AI 设备。由基于 RISC-V 的 SG200X 处理器驱动，它以高能效提供高性能 AI 推理。其模块化设计、先进的视频处理能力和灵活部署支持使其成为各种用例的理想选择，包括安全监控、环境应用和制造业。

- [SONY IMX500](sony-imx500.md)：在配备 IMX500 传感器的 Raspberry Pi AI 相机上优化和部署 [Ultralytics YOLOv8](https://docs.ultralytics.com/models/yolov8/) 模型，实现快速、低功耗的性能。

- [TensorRT](tensorrt.md)：由 [NVIDIA](https://www.nvidia.com/) 开发，这个高性能[深度学习](https://www.ultralytics.com/glossary/deep-learning-dl)推理框架和模型格式优化 AI 模型，在 NVIDIA GPU 上实现加速的速度和效率，确保流畅的部署。

- [TF GraphDef](tf-graphdef.md)：由 [Google](https://www.google.com/) 开发，GraphDef 是 TensorFlow 用于表示计算图的格式，能够在各种硬件上优化执行机器学习模型。

- [TF SavedModel](tf-savedmodel.md)：由 [Google](https://www.google.com/) 开发，TF SavedModel 是 [TensorFlow](https://www.ultralytics.com/glossary/tensorflow) 模型的通用序列化格式，能够在从服务器到边缘设备的广泛平台上轻松共享和部署。

- [TF.js](tfjs.md)：由 [Google](https://www.google.com/) 开发，用于在浏览器和 Node.js 中促进机器学习，TF.js 允许基于 JavaScript 部署机器学习模型。

- [TFLite](tflite.md)：由 [Google](https://www.google.com/) 开发，TFLite 是一个轻量级框架，用于在移动和边缘设备上部署机器学习模型，确保快速、高效的推理，同时占用最小的内存。

- [TFLite Edge TPU](edge-tpu.md)：由 [Google](https://www.google.com/) 开发，用于在 Edge TPU 上优化 TensorFlow Lite 模型，此模型格式确保高速、高效的[边缘计算](https://www.ultralytics.com/glossary/edge-computing)。

- [TorchScript](torchscript.md)：作为 [PyTorch](https://pytorch.org/) 框架的一部分开发，TorchScript 能够在各种生产环境中高效执行和部署机器学习模型，无需 Python 依赖。

## 常见问题

### 什么是 Ultralytics HUB，它如何简化机器学习工作流程？

[Ultralytics HUB](https://www.ultralytics.com/hub) 是一个基于云的平台，旨在使 Ultralytics 模型的机器学习工作流程无缝高效。通过使用此工具，您可以轻松上传数据集、训练模型、执行实时跟踪并部署 YOLO 模型，无需大量编码技能。该平台作为一个集中的工作空间，您可以在其中管理从数据准备到部署的整个机器学习流程。您可以在 [Ultralytics HUB](https://hub.ultralytics.com/) 页面探索关键功能，并通过我们的[快速入门](https://docs.ultralytics.com/hub/quickstart/)指南快速开始。

### 我可以使用 MLFlow 跟踪 Ultralytics 模型的性能吗？

是的，可以。将 [MLFlow](https://mlflow.org/) 与 Ultralytics 模型集成允许您跟踪实验、提高可重复性并简化整个机器学习生命周期。有关设置此集成的详细说明，请参阅 [MLFlow](mlflow.md) 集成页面。此集成对于监控模型指标、比较不同训练运行和高效管理机器学习工作流程特别有用。MLFlow 提供了一个集中平台来记录参数、指标和工件，使理解模型行为和进行数据驱动的改进变得更加容易。

### 使用 Neural Magic 进行 YOLO11 模型优化有什么好处？

[Neural Magic](neural-magic.md) 通过利用量化感知训练 (QAT) 和剪枝等技术优化 YOLO11 模型，产生高效、更小的模型，在资源有限的硬件上表现更好。查看 [Neural Magic](neural-magic.md) 集成页面了解如何实施这些优化以获得卓越的性能和更精简的模型。这对于在计算资源受限的边缘设备上部署特别有益。Neural Magic 的 DeepSparse 引擎可以在 CPU 上提供高达 6 倍的推理加速，使得无需专用硬件即可运行复杂模型成为可能。

### 如何使用 Gradio 部署 Ultralytics YOLO 模型进行交互式演示？

要使用 [Gradio](https://www.gradio.app/) 部署 Ultralytics YOLO 模型进行交互式[目标检测](https://www.ultralytics.com/glossary/object-detection)演示，您可以按照 [Gradio](gradio.md) 集成页面中概述的步骤操作。Gradio 允许您为实时模型推理创建易于使用的 Web 界面，使其成为以用户友好格式展示 YOLO 模型功能的绝佳工具，适合开发人员和最终用户。只需几行代码，您就可以构建交互式应用程序，在自定义输入上演示模型的性能，促进对计算机视觉解决方案的更好理解和评估。
